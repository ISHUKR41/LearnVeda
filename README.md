```bash
cd frontend
```

```bash
cd ../backend

```

#### Step 4 вҖ” Configure Environment

```bash
cd ../frontend
cp .env.example .env.local
# Fill in values вҖ” full reference in the Environment Variables section below
```

#### Step 5 вҖ” Start Database + Cache

```bash
# From project root вҖ” recommended for local dev
docker-compose up -d postgres redis

# PostgreSQL starts with:
# max_connections=200, shared_buffers=256MB, work_mem=4MB
# effective_cache_size=768MB, random_page_cost=1.1
# Slow query logging: > 1000ms

# Redis starts with:
# maxmemory 256mb, allkeys-lru, AOF persistence
```

#### Step 6 вҖ” Run Migrations

```bash
cd frontend
npm run db:migrate
```

### Graceful Shutdown Sequence

When `SIGTERM` or `SIGINT` is received:

```
Step 1 вҶ’ Stop accepting new HTTP connections  (server.close)
Step 2 вҶ’ Stop job scheduler                   (stopScheduler)
Step 3 вҶ’ Flush analytics buffer               (stopAnalyticsFlushTimer)
Step 4 вҶ’ Flush audit log buffer               (stopAuditFlushTimer)
Step 5 вҶ’ Close PostgreSQL pool                (closeDatabasePool)
Step 6 вҶ’ Close Redis client                   (closeRedisClient)
Step 7 вҶ’ Process exit(0)

Timeout: 30 seconds вҶ’ hard kill if graceful exit hangs
```
