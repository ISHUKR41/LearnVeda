# EduQuest Design System

This is the semantic design system for EduQuest, India's #1 gamified learning platform created by Ishu.

## 1. Brand Identity
**Name**: EduQuest by Ishu
**Tagline**: India's #1 gamified learning platform for CBSE & Engineering students.
**Theme**: Gamified, engaging, academic, high-performance, dark-mode native.

## 2. Typography
We use a robust 8-point grid typography system leveraging modern variable fonts.

- **Display/Heading Font**: `Plus Jakarta Sans`, `Space Grotesk`, `Sora`
- **Body/Reading Font**: `Inter`, `Plus Jakarta Sans`
- **Monospace/Code**: `JetBrains Mono`
- **Stats/Numbers**: `Plus Jakarta Sans`

## 3. Color Palette

The platform defaults to a premium **Dark Mode** surface for prolonged study sessions without eye strain. 

### Neutral Surfaces
- **Hero/Background Base**: `#0B1120`, `#0D1117`
- **Surface Level 1**: `#161B22`
- **Surface Level 2**: `#1C2128`
- **Borders**: `#30363D`, `#21262D`

### Brand Colors
- **Primary (Blue)**: `#58A6FF` — Trust, links, primary actions.
- **Secondary (Teal)**: `#39D5CC` — Secondary actions, science/math categories.

### Gamification & State Colors
- **Accent (Amber/Gold)**: `#F0B429` — XP, coins, streak badges, level-ups.
- **Success (Green)**: `#3FB950` — Correct answers, module completion.
- **Danger (Red)**: `#F85149` — Incorrect answers, lost streaks, critical warnings.
- **Warning (Orange)**: `#D29922` — Deadlines, low time warnings.
- **Neon Brights**: Used in interactive components (like Mind Map) for maximum visibility (e.g. `#00ffff`, `#ff00ff`).

## 4. Spacing & Layout
We use a consistent 8-point grid.
- **Base Grid**: 8px (`0.5rem`)
- **Common Margins/Padding**: `16px`, `24px`, `32px`
- **Max Content Width**: `1280px` (Web apps/dashboards)
- **Max Reading Width**: `768px` (For textbook-like topics to maintain optimal line-length)

## 5. UI Components & Borders
- **Border Radius**: 
  - Buttons/Inputs: `12px` (`0.75rem`)
  - Cards: `16px` (`1rem`)
  - Hero Elements: `24px` (`1.5rem`)
- **Shadows & Glows**: We use subtle glowing shadows (`box-shadow: 0 0 24px rgba(..., 0.25)`) to indicate interactive gamified elements.
