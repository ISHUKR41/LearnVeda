# About Route

This folder owns the public `/about` page.

## Files

- `page.tsx` renders the EduQuest mission, platform pillars, and trust content.
- `About.module.css` contains only route-specific styling for the about page.

## Notes

Keep company narrative, product positioning, and public trust information here. Do not add backend database access, authenticated dashboard logic, or unrelated marketing experiments to this route folder.
