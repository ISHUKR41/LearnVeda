# Sign In Route

This folder owns the `/sign-in` page.

## Files

- `page.tsx` renders the login form, validation states, and authentication API submission.
- `SignIn.module.css` contains sign-in form, panel, and responsive styles.

## Notes

Password verification and session creation belong in `/api/auth/sign-in` and backend auth helpers. Keep this page focused on user input and clear error handling.
