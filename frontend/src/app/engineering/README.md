# Engineering Route

This folder owns the `/engineering` learning track and dynamic engineering plan pages.

## Files

- `page.tsx` renders the engineering hub with language and skill tracks.
- `[slug]/page.tsx` renders one engineering learning plan.
- `Engineering.module.css` contains route-specific engineering styles.

## Notes

Keep programming-language and computer-science discovery UI here. Shared plan rendering should stay in `src/components/learning`, and backend-only content fetching should stay in server repositories or services.
